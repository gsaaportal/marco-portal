The Data tab is where you'll find the data layers available for viewing.

  1. Click on a Theme (Jurisdictions and Boundaries, Marine Habitat, etc) to expose its Data Layers
  2. Click a Data Layer to display it on the map

Tour the Data tab

Switching to the Active tab allows you to further manipulate the data activated from the Data tab.

From the Active tab you can adjust the ordering of the layers on the map, as well as their transparency and visibility.

Switch to the Identify tab when you are using the Identify Feature function to query layer data.

In the Identify tab you will see the data returned when you click on a point on the top most active layer.

Tour the Identify tab

Switch to the Query tab when you are using the Polygon Query Tool to determine which layers exist in the polygon.

In the Query tab you will see the same hierarchy as in the Data tab. To see which layers exist in the polygon area, click the parent to show the results. You can click Save Results to save a zip file which contains a map image showing the polygon area, and a PDF which contains the polygon longitudes and latitudes as well as the layers found in the polygon.

Tour the Identify tab

The auto-complete search field helps you locate and view data.

  1. Enter your search term into the search box
  2. Select from the list of auto-complete options
  3. And the theme containing your layer will be expanded to show its location and your layer will be shown on the map

The Topics buttons will add a list of layers related to the selected Topic to the Active tab.

The user can choose which layer or layers for the Topic to enable on the map.

Tour the Active tab

Use the + and - to zoom in and zoom out. Go ahead, Try it!

Alternatively, to zoom in to a particular area on the map:

  1. Hold down the shift key
  2. Then with the mouse button pressed, drag the mouse across the map (the zoom area will be shown on the map)
  3. Release the mouse button when the zoom area is complete

The map includes the following specialized features allowing you to:

  1. Change the underlying Basemaps
  2. Show/Hide Legends associated with visible layers
  3. Show/Hide the Layer Panel
  4. Use Bookmarks to create, store, and share your personalized map views
  5. View the map in Full Screen
  6. Export Map Views for power point presentations, printing, and collaboration
  7. Enable the Identify Feature functionality to retrieve data when clicking a point on the map
  8. Enable the Polygon Query tool to draw a polygon on the map and query to find what layer data is present. Begin drawing the polygon by clicking on the map, terminate the polygon by double clicking last point
  9. Enable the Measure Tool. Click to begin measurement, draw the polyline, then double click to calculate distance

Clicking on the transparency button gives you control over the layer's transparency on the map.

The ordering of the layers on the map mirrors the ordering in the Active panel.

To manipulate the order of the layers simply drag the layer to the desired position in the list of layers.

  1. Click on a layer name
  2. While holding down the mouse button, drag the chosen layer to a different position in the layer ordering
  3. Release the mouse button, and the map layers have been re-ordered, both in the Active panel and on the map
